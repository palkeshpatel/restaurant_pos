## Quick Start

1. **Clone the repository**

   ```bash
   git clone <repository-url>
   cd flutter_app
   ```

2. **Install dependencies**

   ```bash
   flutter pub get
   ```

3. **Run the app**

   ```bash
   flutter run
   ```
